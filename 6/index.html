Berikut adalah hasil saduran saya :
1. Mengingatkan Slogan kampus yaitu Cerdas Berjiwa Luhur.
2. Membuat kembali cara mengisi kehadiran dengan cara membuat file README.md dan file index.html untuk hasil menyadur.
3. Class Diagram digunakan untuk menggambarkan hubungan antar class atau biasanya disebut hubungan antar tabel.
4. Class Diagram terdiri nama Class,Atribut,Method.
5. Nama Class tersebut Nama Tabel. Dimana ketika mengisi nama tabel usahakan jangan ada spasi
6. Atribut diisi dengan field yang menjadi key,nama field,tipe data,lebar atau panjang data.
7. Method yaitu diisi dengan operasi apasaja yang dilakukan pada class atau tabel. Contoh Select/baca,Insert/tambah/simpan,Update/ubah,Delete/hapus,Count/hitung jumlah,Auoto_numbering/nomor otomatis/kode otomatis,Dll.
8. Stereotype digunakan menentukan atau menandakan atribut yang menjadi primary key maupun foreign key.
9. Multiplicity digunakan menentukan tingkat hubungan atau kardinalitas antar class melalui association(garis hubungan antar class).
